## Добавление фото проекта Cowper Road 55

### Источник
[houzz.ie/.../cowper-road-55-pj-vj~7446924](https://www.houzz.ie/hznb/projects/cowper-road-55-pj-vj~7446924) — 17 фотографий проекта (9 из них с пометкой "Before", остальные — после).

### Шаги
1. **Сбор** — пройти по 17 ссылкам `phvw-vp~ID` / `prvw-vr~ID`, вытащить из HTML URL оригинальной картинки `st.hzcdn.com/simgs/.../6.jpg` (или максимального доступного размера). Скачать в `/mnt/documents/houzz-gallery/cowper/`.
2. **Фильтр** — отбросить "Before"-фото (это снимки ДО ремонта, не репрезентативны для портфолио маляра). Останется ~8 финальных фото.
   - *Уточнение ниже — может, всё-таки оставить пары before/after?*
3. **Обработка** — тот же премиум-пресет (Pillow): квадрат 1:1 1200×1200, авто-контраст, тёплый сдвиг, насыщенность +18%, контраст +8%, sharpen. Выход `.webp`.
4. **Загрузка** — продолжить нумерацию: `src/assets/gallery/work-063.webp` … далее, через `lovable-assets create` → `.asset.json`.
5. **Подключение** — ничего менять в `gallery.tsx` НЕ нужно: там стоит `import.meta.glob` по `assets/gallery/*.webp.asset.json`, новые файлы автоматически появятся в общей сетке.

- Уже добавленные 62 фото
