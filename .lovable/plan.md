## Добавление 11 фото проекта Vale View Avenue

### Что найдено
[Vale View Avenue](https://www.houzz.ie/hznb/projects/vale-view-avenue-pj-vj~7479085) — **11 фотографий "After"**, все по ссылкам `/products/...prvw-vr~ID`. Метки "Before" нет.

Прямой URL изображения находится в HTML как `st.hzcdn.com/fimgs/{hash}_{num}-w...-h...-home-design.jpg` (≈1132×1444 px) — этого достаточно для квадрата 1200×1200.

### Шаги
1. **Скачать** все 11 фото в `/mnt/documents/houzz-gallery/vale-view/` через `curl` + regex на главный fimgs URL страницы товара. Преобразовать в `simgs/{hash}_16-{num}/home-design.jpg` для максимального размера, с фолбэком на исходный fimgs URL.
2. **Дедуп** по хэшу относительно уже загруженных 62 фото в `src/assets/gallery/`.
3. **Обработка** тем же премиум-пресетом (Pillow): квадрат 1:1 1200×1200, авто-контраст, тёплый сдвиг, +18% насыщенности, +8% контраст, sharpen. Сохранить `.webp`.
4. **Загрузка** — продолжить нумерацию `work-063.webp` … далее. Создать `.asset.json` через `lovable-assets`. Удалить исходные `.webp` после.
5. **Подключение** — никаких изменений в `gallery.tsx` не нужно: `import.meta.glob` подхватит автоматически.
